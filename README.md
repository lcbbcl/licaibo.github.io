# 使用说明

> * 修改_config.yaml及about.md文件，以变更个人信息。
> * 配置读取说明: ${site.data.author.workHistory}，前面site.data是国定的，author是author.yml的文件名，workHistory是文件名里面的内容。
> * 修改CNAME文件，以绑定自己的域名。
> * 删除_posts下文章，换成你自己的。
> * 最后，push到你自己的博客Repo~
